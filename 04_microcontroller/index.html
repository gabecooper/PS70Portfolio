Week 4: Microcontroller Programming

For my microcontroller programming week, I fulfilled a long-held childhood goal: make a laser security maze! First, I plotted out the wiring in the diagram attached in the carousel (couldn't find ESP32 so I used the generic microcontroller and the symbol unattached for GND).

And then I wired it all together, using the soldering iron, heat shrinks, and the wire-stripper.

After that I calibrated the LDR against the laser by using the serial monitor to keep an eye on the output level using analog read… and it turns out that with a 10k resistor the LDR is just maxed out which happens to work perfectly fine for this project.

And then, finally, I wrote the code that will turn on the red light if the laser beam is tripped (see video).

This project was the first time that I've ever diagrammed a circuit before, and I definitely see the appeal of knowing exactly what will be wired where before I start using the soldering iron and connecting things. Altogether, this project was a blast but definitely felt elementary especially after having learned so much more about wiring and soldering in the weeks after this assignment was initially due. Going forward, I think attaching a light diffuser like tissue paper would help make this even more reliable so that it's not so hard to position the laser perfectly and so that minor vibrations don't trigger the light.

Arduino Code
const int LDR_PIN = A0;
const int LED_PIN = D1;

void setup() {
  pinMode(LED_PIN, OUTPUT);
  pinMode(LDR_PIN, INPUT);
}

void loop() {
  if (analogRead(LDR_PIN) < 4000) {
    digitalWrite(LED_PIN, HIGH);
  } else {
    digitalWrite(LED_PIN, LOW);
  }
}
